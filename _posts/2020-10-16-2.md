#### cin 정수로 입력받기
원하는 input 형태: 'start(int) (space) length(int)'
오류 제어: 각 변수 타입이 int 가 맞는지, 3개 이상의 인풋이 들어오는지

##### try 1: cin >> start >> length;
이렇게 되면 만약 '3 1 2' 로 입력했을 때 입력 버퍼에 2가 남게 된다.
